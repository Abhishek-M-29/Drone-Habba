# Xcelerate 2026

A modern event website for **Xcelerate 2026** - an aerospace and aviation competition hosted by the Aeronautical Engineering Department at Acharya Institute of Technology, Bangalore.

## Event Details

- **Date:** March 27, 2026
- **Venue:** Acharya Institute of Technology, Soladevanahalli, Bangalore

## Events

1. **Drone Competition** - Autonomous drone flying with geofencing (waypoints). Max weight 2kg including payload.
2. **Parachute Challenge** - Materials provided on-site. Come with an idea and build within given time.
3. **Aircraft Investigation** - Analyze a random aircraft accident video and investigate the cause.
4. **Design Parameters** - Design an aircraft based on given parameters for any given objective.

## Tech Stack

- **React** - UI framework
- **Vite** - Build tool
- **Tailwind CSS** - Styling
- **Framer Motion** - Animations
- **Lucide React** - Icons

## Getting Started

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```
