# AeroXcelerate 2026

A modern event website for **AeroXcelerate 2026** — an aerospace and aviation competition hosted by the Department of Aeronautical Engineering at Acharya Institute of Technology, Bangalore. Organized by **The Big O** tech club.

## Event Details

- **Date:** March 27, 2026
- **Venue:** Acharya Institute of Technology, Soladevanahalli, Bangalore

## Events

1. **Drone Competition** — Autonomous drone flying with geofencing and waypoints. Max weight 2kg including payload.
2. **Egg Parachute Competition** — Materials provided on-site. Come with a design concept and build within the allotted time.
3. **Aircraft Investigation** — Analyze a random aircraft accident video and investigate the probable cause.
4. **Aircraft Design** — Design an aircraft based on given engineering parameters for a specified objective.
5. **Aircraft Poster Presentation** — Prepare and present a research poster on any aviation/aerospace topic.

## Tech Stack

- **React** — UI framework
- **Vite** — Build tool
- **Tailwind CSS** — Styling
- **Framer Motion** — Animations
- **Lucide React** — Icons

## Sections

- **Hero** — Animated landing with countdown timer and drone wireframe SVG
- **About** — Event overview, value propositions, bento stats grid, parallax background, hosted-by cards
- **Events** — Featured card, event grid with modals, stacked evaluation criteria bar
- **Timeline** — Event day flow
- **Registration** — Registration form and details
- **FAQ** — Frequently asked questions

## Getting Started

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```
